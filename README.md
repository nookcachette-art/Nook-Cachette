## 🖼 画像を更新する手順

### 1. 画像ファイルを差し替える

`images` フォルダの中にある画像を、新しい画像で上書きしてください。

| ファイル名 | 用途 |
|---|---|
| `gallery-1.jpg` | ギャラリー画像1（大きく表示） |
| `gallery-2.jpg` | ギャラリー画像2 |
| `gallery-3.jpg` | ギャラリー画像3 |
| `gallery-4.jpg` | ギャラリー画像4 |
| `gallery-5.jpg` | ギャラリー画像5 |

※ ファイル名は変えないでください。  
※ 画像は1200px × 900px 以上、500KB 以下を推奨します。

---

### 2.「更新する.bat」をダブルクリック

フォルダ内の **「更新する.bat」** をダブルクリックしてください。

```
変更内容が表示されます
 ↓
「Y」を押してEnter
 ↓
自動でアップロード完了
```

---

### 3. 2〜5分で反映される

https://shunsuke2121.github.io/Nook-Cachette/

※ 反映されない場合はブラウザを再読み込み（F5）してください。

---

## ❓ Q&A

**Q: 「更新する.bat」がエラーになる**  
A: Git がインストールされていない可能性があります。以下からインストールしてください。  
　 https://git-scm.com/download/win

**Q: 画像が5枚以上追加したい**  
A: 開発者に連絡してください。

**Q: テキストを変えたい**  
A: 開発者に連絡してください。`index.html` の編集が必要です。
