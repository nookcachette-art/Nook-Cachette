## 📸 画像の更新方法

このサイトは **GitHub** で管理されています。  
クライアントは画像の追加・変更だけで、サイトが自動で対応します。

### 🖼 ギャラリー画像の更新手順

1. **GitHub にログイン**  
   https://github.com/shunsuke2121/Nook-Cachette

2. **`images` フォルダを開く**  
   リポジトリ内の `images` → `Upload files` をクリック

3. **画像をアップロード**
   - ファイル名は以下の名前で上書きしてください：

   | ファイル名 | 用途 | サイズ目安 |
   |---|---|---|
   | `gallery-1.jpg` | ギャラリー画像1（大） | 1200x900px 以上 |
   | `gallery-2.jpg` | ギャラリー画像2 | 1200x900px 以上 |
   | `gallery-3.jpg` | ギャラリー画像3 | 1200x900px 以上 |
   | `gallery-4.jpg` | ギャラリー画像4 | 1200x900px 以上 |
   | `gallery-5.jpg` | ギャラリー画像5 | 1200x900px 以上 |

4. **アップロード完了**  
   約2〜5分で https://shunsuke2121.github.io/Nook-Cachette/ に反映されます

   ※ ブラウザのキャッシュがある場合は、Ctrl+Shift+Delete（Windows）または Command+Shift+Delete（Mac）でキャッシュをクリアしてください

## ❓ Q&A

**Q: 画像サイズはどのくらいがいい？**  
A: 1200px × 900px 以上の高解像度画像を推奨します。ファイルサイズは 500KB 以下にしてください。

**Q: JPG 以外の形式でもいい？**  
A: はい。PNG、WebP なども対応しています。ただしファイル名は `.jpg` で統一してください。

**Q: 画像の順序を変更したい**  
A: ファイル名の番号（gallery-1, gallery-2...）で自動的に順番が決まります。

**Q: 画像が反映されない**  
A: 
1. ブラウザを再読み込み（F5キー）
2. ブラウザキャッシュをクリア
3. 10分以上経過したか確認

**Q: 急いでいて、すぐに反映させたい**  
A: GitHub Pages は 2〜5 分で反映されます。急ぐ場合は開発者に連絡してください。

**Q: 画像を削除したい**  
A: `images` フォルダから該当ファイルを削除（Delete）してください。サイトから自動的に消えます。
